## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or above)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/mark-it.git
   cd mark-it
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) to view the app.

## Usage

- **Add Bookmarks:** Save new bookmarks with custom titles, descriptions, URLs, and tags.
- **Search:** Use the search bar to find bookmarks by title, description, or tag.
- **Filter:** Click tags in the sidebar to filter bookmarks.
- **Sort:** Change sorting order from the filter section.
